arXiv Categories Taxonomy

In this module we are defining the main and the sub global dictionaries that defines the top, and sub, categories of the arXiv taxonomy. The key are the categories id that should be used to interact with the ReadNext CLI commands. The value is a human readable description of each category.

Exists

The exists function is used to check if a category exists in the taxonomy. It returns True if the category exists, False otherwise.

exists

 exists (category:str)

Check if a category, or a subcategory exists in the arXiv taxonomy.

Tests

assert exists("cs") == True
assert exists("all") == True
assert exists("cs.AI") == True
assert exists("cs.ai") == False
assert exists("cs.FOO") == False
assert exists("foo") == False
assert exists("alll") == False
